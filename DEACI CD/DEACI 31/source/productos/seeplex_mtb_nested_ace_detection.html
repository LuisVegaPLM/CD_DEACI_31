SEEPLEX® MTB NESTED ACE DETECTION

INDICACIÓN DE USO:

La prueba Seeplex® MTB Nested ACE Detection está diseñada para la detección de Mycobacterium tuberculosis (MTB), IS6110 y MPB64, empleando PCR anidada.

Analitos:

• Detección de Mycobacterium tuberculosis.

Muestras:

• Tejido.

o Tejido fresco.

o Tejido embebido en parafina.

• Esputo.

• Lavado bronquial.

CARACTERÍSTICAS:

• Detección de Mycobacterium tuberculosis por PCR anidada a partir de una variedad de muestras, incluidas muestras de tejido embebido en parafina.

• Provee una solución para la extracción de DNA. Proporciona un reactivo para la detección de DNA (incluye solución de extracción).

• Sistema para la interpretación de resultados.

• Sistema para prevención de contaminación.
