COAGULACIÓN, REACTIVOS PARA


APTT-XL. Tromboplastina líquida lista para usarse en el tiempo de tromboplastina parcial activada. Altamente sensible a la deficiencia de los factores de coagulación de la vía intrínseca, inhibidores de lupus, para el monitoreo de la terapia con heparina. ACCUTRACK.

COAGULACIÓN. Productos para coagulación, para uso de forma manual y automatizada. Thromboplastin-DS; APTT-XL; Thrombin Time Reagent; Plasma Control de Coagulación, Nivel 1; Plasma Control de Coagulación, Nivel 2; Plasma Control de Coagulación, Nivel 3; Fibrinogen Assay Set. ACCUTRACK.

FIBRINOGEN ASSAY SET. Es una prueba para la determinación rápida y cuantitativa de fibrinógeno en plasma (método de clauss). El kit contiene trombina bovina, plasma de referencia de fibrinógeno y buffer salino de imidazol. 100 pruebas. ACCUTRACK.

PLASMA CONTROL DE COAGULACIÓN, NIVEL 1. Plasma humano normal, se usa como control para monitorear el desempeño de las pruebas de rutina de coagulación. 10 x 1 ml. ACCUTRACK.

PLASMA CONTROL DE COAGULACIÓN, NIVEL 2. Plasma humano moderadamente anormal (bajo) se usa como control para monitorear el desempeño de las pruebas de rutina de coagulación. 10 x 1 ml. ACCUTRACK.

PLASMA CONTROL DE COAGULACIÓN, NIVEL 3. Plasma humano significativamente anormal (alto). Se usa como control para monitorear el desempeño de las pruebas de rutina de coagulación. 10 x 1 ml. ACCUTRACK.

THROMBIN TIME REAGENT. Reactivo de trombina para la determinación del tiempo de trombina (TT) en forma manual o automatizada. 10 x 1 ml. ACCUTRACK.

THROMBOPLASTIN-DS. Tromboplastina de alta sensibilidad (isi = 1.0) de cerebro de conejo y buffer, para el tiempo de protrombina y para las pruebas de los factores de coagulación de la vía extrínseca (monitoreo de pacientes anticoagulados). ACCUTRACK.
